项目经历

自然语言处理
该项目处理的对象为一本信息科技读物（介绍信息科技中的名词和技术），目的是得出自写率，分析文本抄袭情况。

  • 根据名词实时爬取百科解释实时对比。
  • 使用 TextRank 提取关键字，摘要。
  • 基于互信息和左右信息熵提取短语。

IoT 的服务器端
该项目为参加比赛的成果。客户端位于树莓派，服务器端实现接收客户端发送（使用 socket）的图片，并根据网络情况将图片拼接位视频并存储。最后基于视频进行移动物体识别。

  • 将图片转化为.avi 视频保存。
  • 缓冲区管理。
  • 定制 UDP 数据包。

Java餐饮管理系统
该项目是一个纯 Java 的餐饮管理系统，涉及到餐桌管理、菜品管理、会员管理等等。数据库使用了 SQLserve。
