SwiftUI官方教程（一）

Created By: jas chen
Last Edited: Feb 09, 2020 9:17 PM

版本要求：macOS Catalina 10.15 、Xcode 11

创建项目

打开 Xcode，创建新的 Xcode 项目，选择 iOS 平台，选择 Single View App，然后点击下一步。

输入项目名称 LandMarks，选择使用 SwiftUI 构建用户界面，点击下一步，选择将项目保存到哪里。

在项目的导航栏里，选择 ContentView.swift

swiftUI 默认会声明两个结构( Structure ) , 第一个结构遵循了 View 协议，用来描述本视图的内容和样式。第二个结构声明了本视图的预览（即 Xcode 右边的 Canvas 中显示的内容）。
